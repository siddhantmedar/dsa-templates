# DSA Templates

A collection of Data Structures and Algorithms templates implemented in Python. These are ready-to-use templates for competitive programming and coding interviews.

## Contents

| File | Description |
|------|-------------|
| `dfs_plain.py` | Basic Depth-First Search implementation |
| `dfs_on_matrix.py` | DFS traversal on a 2D matrix |
| `dijkstra.py` | Dijkstra's shortest path algorithm |
| `bellman_ford.py` | Bellman-Ford shortest path algorithm |
| `prims.py` | Prim's Minimum Spanning Tree algorithm |
| `kruskal.py` | Kruskal's Minimum Spanning Tree algorithm |
| `kosarajus_algorithm.py` | Kosaraju's algorithm for Strongly Connected Components |
| `topological_sort.py` | Topological sorting |
| `topological_sort_dfs.py` | Topological sort using DFS |
| `toplogical_sort_bfs.py` | Topological sort using BFS (Kahn's algorithm) |
| `word_ladder.py` | Word Ladder problem (BFS) |
| `word_ladder_2.py` | Word Ladder II problem |
